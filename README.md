# Sim!Bora 

App para descobrir rolês, eventos e experiências locais acessíveis.

## Funcionalidades (MVP)
- Login e Cadastro
- Listagem de eventos
- Detalhes do evento
- Confirmação de presença (RSVP)
- Perfil do usuário

## Tecnologias
- React Native (Expo)
- Firebase (Auth + Firestore)
- JavaScript/JSX
